# Pastel UI

Pastel UI is a boutique React UI component library. It's focused on aesthetic, clean visuals.

Pastel UI is designed for web applications that use strong photography, video, or bold graphics.
Its subtle, elegant color palette lets your visual content shine.

List of components it provides:

1. Accordion
2. Button
3. Card
4. Checkbox
5. Text Input
6. Radio Button
7. Text Area[^1]
8. Dropdown[^2]
9. Date and Time Picker[^2]

[^1]: In progress
[^2]: To do

## Running the project for local development

Prerequisites:

1. Node.js
2. Git

Steps to run the project:

1. Clone this repository (`git clone https://github.com/nataliakuntic/pastel-ui.git`) to a location on your hard drive.
2. `cd` into the project directory.
3. Run `npm install`.
4. Run `npm run dev`.
5. The project should now be running at `http://localhost:5173`
